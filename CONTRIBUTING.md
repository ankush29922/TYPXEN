# Contributing

Issues and narrowly scoped patches that improve the documented original
ESP32-CAM target, tests, setup tools, or documentation are welcome. Do not submit
private keys, credentials, licence tokens, customer data, recorder data,
captured source, device identifiers, network identifiers, APKs, or build output.

Before proposing a change:

1. Confirm it does not alter HID timing, cadence behavior, cryptography,
   licensing, or the Protocol V1 contract without an explicit validation plan.
2. Run the relevant host tests and `git diff --check`.
3. Use placeholders for ports, paths, SSIDs, addresses, and device identifiers.
4. State the exact board and host environment used for any hardware result.
5. Do not claim support for an untested board.

No reuse licence has been selected. A contribution does not change the
repository's licensing status unless the owner expressly adopts a licence.
